# Python + GIS Data Workflow

## Objective
Demonstrate how Python can be used to clean and prepare location-based data before importing it into QGIS for mapping and analysis.

## Project Overview
This project simulates a real-world GIS workflow where raw data is first processed in Python, then visualized in QGIS.

## Tools Used
- Python
- Pandas
- QGIS
- CSV data

## Workflow
1. Load raw location data in Python  
2. Clean and standardize the dataset  
3. Export a GIS-ready CSV file  
4. Import the cleaned file into QGIS  
5. Visualize the points on a map  

## Outcome
This project demonstrates how Python and GIS tools can be combined for spatial data processing and visualization.
